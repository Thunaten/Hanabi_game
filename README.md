# Hanabi_game

Проект написан для проверки навыков ООП.

Игра, по сути, представляет собой пасьянс. Целью является составление пяти цепочек карт, значениями от 1 до 5 в порядке возрастания. При розыгрыше карты, которую нельзя добавить в цепочку - карта сбрасывается. Игра заканчивается если собрать все цепочки или если на руке и в колоде не останется карт.

Внимание: В колоде 5 цветов, по 10 карт каждого цвета: 3 карты номиналом «1», по 2
карты «2», «3» и «4», 1 карта «5».
